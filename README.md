# Convert JSON

Simple Form to convert JSON.

# Feature
- Input Form
- Code Editor
- Download as JSON
- Download as PDF

**CODE**
```
{
	"object": {
		"key1": "Value 1",
		"key2": "Value 2",
		"key3": "Value 3",
		"object2": {
			"key4": "Value 4"
		}
	}
}
```
